<p>
  A stock market inspired web application that allows users to search for the
  latest market information on any publicly tradable US stock listed on the{" "}
  <span>Investors Exchange</span> (IEX). All of the market data is retrieved
  from <span>IEXCloud API</span> using the built-in JavaScript fetch API. The
  web app includes an interactive intraday price movement chart created with{" "}
  <span>ApexCharts.js</span>.
</p>
